# Retail Sales Dashboard (Excel)

## Project Overview
This project analyzes retail sales data using Excel.

## Dataset
Dataset contains 20,000 rows including:
- Order Date
- Customer Name
- State
- Category
- Sales
- Quantity
- Profit

## Tools Used
- Microsoft Excel
- Pivot Tables
- Slicers
- Charts
- Map Visualization

## Key Insights
- Technology category has the highest sales
- California generates the highest revenue
- Some products generate negative profit

## Dashboard
The dashboard allows interactive filtering using slicers.
